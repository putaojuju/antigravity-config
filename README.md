# Antigravity Rules & Prompts

这是为 **Antigravity**（强大的代理型 AI 编程助手）量身定制的系统规则与提示词配置。本仓库旨在分享一套高效的、面向 **No-Code 开发者** 的工作流协议，通过结构化的指令大幅提升 AI 的执行精度与响应质量。

## 🚀 核心特色

- **ACIP 标准执行**：采用“静默状态流转”协议，最小化冗余输出，最大化推理效率。
- **多模型协同逻辑**：
  - **Gemini 3 Flash**：极致速度，处理原子化技术任务。
  - **Gemini 3.1 Pro / Claude Sonnet**：深度思考，负责顶层架构与任务拆解。
  - **Claude Opus**：高可靠性，攻克复杂边界案例。
- **No-Code 友好**：优先采用高层抽象解释，无需用户具备深厚代码背景即可直接使用。
- **中文原生支持**：深度优化中文交互体验。

## 📂 仓库内容

- `GEMINI.md`: 核心系统提示词文件，包含完整的角色定义、执行协议与行为逻辑。

## 🛠️ 如何使用

1. **直接引用**：在 Antigravity 界面中，将 `GEMINI.md` 的内容添加到你的系统提示词（System Prompt）或项目规则空间中。
2. **学习参考**：如果你正在编写自己的 AI Agent 规则，本仓库的结构化表达方式具有极高的参考价值。
